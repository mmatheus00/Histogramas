# Distribuição de Frequência e Histogramas
Exercícios realizados com base no Capítulo 2 do livro "Introdução a Estatística" 11ª Edição, de Mario F. Triola. 

Venho desenvolvendo meus fundamentos estatísticos com base nesse livro a aproximadamente um mês.
No capítulo dois, o objetivo é aprender a organizar, resumir e e representar os dados de forma mais conveniente e significativa possível. E além disso, analisar os dados e entender o que eles estão nos dizendo.

Nesse quesito, a Distribuição de Frequência é muito útil para organizar e resumir os dados, formando uma tabela. Essa tabela nos mostra como o conjunto de dados esta dividido entre as diversas categorias (ou classes), e pode nos dar informações sobre valores atípicos (outliers), se existem características dos dados que mudam com o tempo, entre outros.

Já o Histograma, é um gráfico de barras que pode ser utilizado para representar a distribuição de frequência, com a escala horizontal representando as categorias, e a vertical representando as frequências de cada categoria.

Em suma, nosso principal objetivo usando essas ferramentas é sempre aprender mais sobre o conjunto de dados estudado, e quais informações relevantes conseguimos extrair deles para se resolver um problema específico.
